# BritishMysteries

This is an online archive for BM documents.

The intention is for this to capture as much BM material as people have, this includes elements of development which proved to be 'dead ends'.

Documents which are considered BM sympathetic can also be stored here.

If people wish to provide some additional thoughts with submitted documents I'll look to include these with the documents. This may include when it was produced, what the environment was, intention was, reflection now, etc.

This is only to be shared with householders and those who have a long lasting interest and engagement in BM.

With that being said this link is public and judgement or appropriateness will be required by those sharing the link.
